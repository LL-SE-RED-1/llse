帮助手册（系统管理员版）

登录

  • 在浏览器中输入本系统网址即可进入登录界面。
  • 功能：验证身份以进入教务管理系统。
  • 操作方法
    • 输入正确的学号/教工号。
    • 输入相应的密码。
    • 选择正确的用户类型。
    • 点击登录，验证成功即可登录，否则会出现错误提示信息。
  • 输入格式
    • 学号/教工号：10位纯数字。
    • 密码：最多30位，最少6位。初始密码与学号/教工号相同。

重置密码

  • 在登录界面，点击右上角重置密码即可进入
  • 功能：重置用户的登录密码
  • 操作方法：
    • 用户输入自己的学号，系统将会发送一封带修改密码链接的邮件到用户的校网邮箱。
    • 用户进入校网邮箱查收邮件（可能在垃圾邮件中），点击邮件中的链接，跳转到修改密码界面。
    • 在修改密码界面两次输入新密码即可修改。界面会显示修改成功或失败的信息。
  • 输入格式：
    • 学号：10位纯数字
    • 新密码和确认新密码文本框中的输入必须相同。

修改密码

  • 登录成功后，进入主界面，点击右上角修改密码即可进入。
  • 功能：修改用户的登录密码。
  • 操作方法：
    • 输入正确的原密码。
    • 两次输入新密码。
    • 点击确认，提交修改申请，界面会显示修改成功或失败的信息。
    • 点击取消，放弃之前的输入，回到欢迎界面。
  • 输入格式：
    • 密码：格式与登录时的格式要求相同。
    • 新密码和确认新密码文本框中的输入必须相同。

注销

  • 登录成功后，进入主界面，点击右上角注销即可注销。
  • 功能：注销当前用户，重新回到登录界面。

添加学生

  • 登录成功后，点击左侧信息管理，再点击右侧添加学生图标，即可进入相应界面。
  • 功能：添加学生用户。
  • 操作方法：
    • 输入学生的基本信息（名称后带红色*的为必填项目）
    • 点击提交，添加该学生，界面显示添加成功或失败的信息。
    • 点击返回，返回之前的界面。
  • 输入格式：
    • 学号：10位纯数字
    • 名字、名族：中文或英文
    • 邮箱：正确的邮箱格式
    • 生日：正确的日期格式，如19990101
    • 手机号：11位纯数字
    • 学院、学系、年级、班级：下拉选项框

搜索学生

  • 登录成功后，点击左侧信息管理，再点击右侧搜索学生图片，即可进入相应界面。
  • 功能：搜索学生信息。
  • 操作方法：
    • 在右侧的下拉框中选择搜索的字段。
    • 在右侧的输入框中输入该字段的搜索条件。
    • 键入回车即可搜索。如果未输入任何条件点击回车，界面显示系统中所有的学生列表。
    • 搜索结果会以列表形式罗列在页面上，点击任意一条的查看详情，即可进入学生信息管理页面，查看学生的个人信息。
  • 输入格式：
    • 编号：10位纯数字
    • 姓名：中文或英文
    • 学院：中文或英文

管理学生信息

  • 在搜索界面上点击任意一条搜索结果右侧的查看详情，即可进入相应界面。
  • 操作方法：
    • 修改相应输入框的值
    • 点击提交，即可提交修改，之后界面会显示修改成功或失败的信息。
    • 点击删除，即可删除该学生，之后界面会显示删除成功或失败的信息。
    • 点击返回，返回之前的界面。
  • 输入格式：
    • 学号和名字不可修改，其余均为可修改
    • 具体字段的格式见上。
